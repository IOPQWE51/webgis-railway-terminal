# 🚂 日本旅行全息终端 · WebGIS 主干地形图系统

**Japanese Railway Holographic Terminal · WebGIS Base Map System**

> 一个基于 **WebGIS 技术**的个人定制版**日本全境旅行漫游系统**，集成了**卫星地图导览**、**气象雷达覆盖**、**铁道网线追踪**、**实体坐标定位**与**次元圣地巡礼**功能的"赛博控制台"。

---

## ⚡ 核心功能模块 (Core Features)

### 🗺️ **地形导航与搜索系统**
- ✅ **Esri 高清卫星地形图**（拓扑/卫星/暗黑三模式）
- ✅ **实时搜索导航**（SearchNavEngine）- OSM Nominatim 地理编码
- ✅ **平滑飞跃动画**（FlyTo）- 1.5 秒内快速定位任意地点
- ✅ **聚合点智能缩放**（Marker Cluster）- 单击聚合数字会自动缩小，所有点保留在视图内
- ✅ **双引擎导航集成** - Google Maps + Yahoo!乗換案内（日本换乘）

- ✅ **浏览器 Geolocation API** - 获取用户实时 GPS 位置

- ✅ **真实位置镜头锁定** - 地图自动飞跃到玩家当前位置
- ✅ **精度显示**（±5米误差）

### 🌦️ **气象雷达与环境矩阵**
- ✅ **OpenWeatherMap 实时降雨/云层覆盖**
- ✅ **全球 8 大枢纽气象节点**（北京/东京/新加坡/伦敦等）
- ✅ **透明度实时调整**（0-100% 滑动）
- ✅ **温度/天气图标动态渲染**

### 🚃 **铁道追踪与骨架系统**

## 🎯 核心交互流程 (Interaction Flows)

### **工作流 #1: 搜索地点并导航**
```
用户输入地点名称
    ↓
SearchNavEngine 调用 Nominatim API
    ↓ [获得坐标]
地图 FlyTo 平滑飞跃（1.5s）
    ↓
打下红色脉冲标靶 🎯
    ↓ [用户点击导航按钮]
┌─────────────────────────┐
│ 📍 Google Maps 路线规划  │
│ 🚃 Yahoo! 换乘方案      │
└─────────────────────────┘
```

### **工作流 #2: 注入真实坐标**
```
用户点击"注入真实坐标"按钮
    ↓
浏览器请求权限 → 用户授予 GPS 访问权
    ↓
获取 { latitude, longitude, heading?, accuracy }
    ↓
打下蓝色呼吸脉冲标靶 🔵
    ↓
地图自动锁定并飞跃到玩家位置
    ↓
显示：📍 您的位置 | 35.6895°N, 139.6917°E ±5m
    ↓ [可继续导航]
集成 Google/Yahoo! 周边导航
```

### **工作流 #3: 聚合点缩放与炸开**
```
用户点击地图上的聚合点气泡（显示数字）
    ↓
触发 clusterclick 事件
    ↓
fitBounds() 计算包围框
    ↓
地图平滑缩放到刚好装下所有点 (padding: [50,50])
    ↓
聚合数字自动减少 ✨
    ↓ [继续点击]
重复步骤 2-4，逐步放大
    ↓ [到达最高缩放]
spiderfy() → 烟花式爆炸散开 🎆
    ↓
显示所有单个标记
```

### **工作流 #4: 测距雷达**
```
用户点击"启动测距雷达"按钮
    ↓ [按钮变黄，瞄准中...]
用户在地图上点击第一个点
    ↓
地图显示蓝色圆圈标记 ○
    ↓
用户在地图上点击第二个点
    ↓
自动计算两点间直线距离
    ↓
显示："X.XX km" + 清除按钮
```

## 🔧 常见问题排查 (Troubleshooting)

### Q1: 页面空白（白屏）
```bash
# 检查步骤：
1. 按 F12 查看控制台是否有红色错误
2. 确认 node_modules 文件夹存在
3. 检查是否有未关闭的上一个 npm run dev 进程
4. 清空缓存重新安装：
   rm -rf node_modules package-lock.json
   npm install
   npm run dev
```

### Q2: 地图无法加载
```javascript
// 可能原因与解决方案：
1. ❌ Leaflet CDN 被墙
   → 使用国内镜像或 VPN
   
2. ❌ ESRI API 超时
   → 检查网络连接，切换到高清卫星模式再切回
   
3. ❌ 浏览器不支持 WebGL
```
