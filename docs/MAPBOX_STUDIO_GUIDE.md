# 🎨 Mapbox Studio 霓虹金铁路样式创建指南

## 🎯 目标效果

创建一个自定义地图样式，将所有铁路路线渲染为"脉冲琥珀金" (#fbbf24) 霓虹效果。

---

## 📋 第1步：打开 Mapbox Studio

1. 访问：https://studio.mapbox.com/
2. 登录你的账号（iopqwe51）
3. 点击 **New style**

---

## 📋 第2步：选择基础模板

选择 **"Mapbox Streets"** 作为基础样式：
- ✅ 详细的街道数据
- ✅ 包含铁路图层
- ✅ 适合城市导航

点击 **Use this template**

---

## 📋 第3步：定位铁路图层

### 3.1 打开图层编辑器

1. 点击左侧菜单的 **Layers** 图标（📚）
2. 在搜索框输入：`rail`
3. 找到以下图层：
   - `road rail` - 铁路主线
   - `road rail narrow` - 窄轨铁路
   - `bridge rail` - 铁路桥梁
   - `tunnel rail` - 铁路隧道

### 3.2 验证铁路数据

1. 地图移动到有铁路的区域：
   - 日本东京：`139.6503, 35.6762`
   - 日本大阪：`135.5023, 34.6937`
   - 美国纽约：`-74.0060, 40.7128`

2. 确认看到：
   - 浅灰色的铁路线
   - 车站标记

---

## 📋 第4步：定制铁路样式

### 4.1 选中铁路图层

1. 点击 `road rail` 图层
2. 右侧出现 **Style** 面板

### 4.2 修改颜色

在 **Style** 面板中：

**Line Color（线条颜色）**：
```
点击颜色选择器 → 输入：#fbbf24
（琥珀金色）
```

**Line Width（线条宽度）**：
```
Base: 2.0
（比默认稍宽，突出显示）
```

**Line Opacity（线条透明度）**：
```
Base: 0.9
（90%不透明度，保留细节）
```

### 4.3 添加发光效果（可选高级效果）

在 `road rail` 图层上：

**添加外发光**：
1. 点击 **Layer** → **Duplicate layer**（复制图层）
2. 重命名为：`road rail - glow`
3. 调整样式：
   - Line Color: `#fbbf24`
   - Line Width: `4.0`（更宽）
   - Line Opacity: `0.3`（半透明）
   - Line Blur: `1.0`（模糊效果）

**效果**：铁路会有发光效果 ✨

### 4.4 应用到所有铁路图层

重复上述步骤，修改其他铁路图层：
- `road rail narrow`
- `bridge rail`
- `tunnel rail`

**隧道特殊处理**：
```
对于 tunnel rail：
Line Color: #b45309（深琥珀金，模拟地下）
Line Dasharray: 2, 2（虚线效果）
```

---

## 📋 第5步：定制车站样式（可选）

### 5.1 找到车站图层

搜索：`station` 或 `transit`

找到图层：`poi station`

### 5.2 修改车站标记

**Circle Color（圆点颜色）**：
```
Base: #fbbf24
```

**Circle Radius（圆点半径）**：
```
Base: 6.0
（更大更显眼）
```

**Circle Stroke Width（边框宽度）**：
```
Base: 2.0
```

**Circle Stroke Color（边框颜色）**：
```
Base: #1e293b
（深空灰色边框）
```

---

### 6.1 找到水体图层

搜索：`water`

找到图层：`water`

### 6.2 修改水体颜色

**Color**：
```
点击颜色选择器 → 输入：#1e293b
（深空灰色）
```

**效果**：水体呈现深色，与霓虹金铁路形成对比 🌊

---

## 📋 第7步：调整背景色（可选）

### 7.1 找到背景图层

搜索：`background`

找到图层：`background`

### 7.2 修改背景色

**Color**：
```
深色模式：#0f172a（深空黑）
浅色模式：#f8fafc（浅灰白）
```

**推荐**：使用深色模式，霓虹金效果更明显 ✨

## 📋 第8步：保存并发布样式

### 8.1 保存样式

```
Tactical Terminal - Neon Amber Gold
```

### 8.2 发布样式

点击 **Publish** → Mapbox 会生成一个 **Style URL**

**示例 URL**：
```
mapbox://styles/iopqwe51/xxxxxxxxxxxxxxxx
```

**⚠️ 重要**：复制这个 Style URL！前端集成时会用到。

## 🔧 常见问题

### Q1: 找不到铁路图层？
**A**:
- 确认使用的是 "Mapbox Streets" 模板
- 尝试搜索 `road`, `transit`, `rail`
- 移动地图到有铁路的区域

### Q2: 颜色不生效？
**A**:
- 检查图层是否被其他图层覆盖
- 调整图层顺序（拖动图层）
- 检查透明度设置

### Q3: 样式发布后看不到效果？
**A**:
- 清除浏览器缓存
- 检查 Style URL 是否正确
- 等待几分钟（Mapbox CDN 同步）
