## ⚠️ 申请注意事项

### ✅ 好的做法
- 说明应用用途清晰
- 提供真实的使用场景
- 选择合适的免费计划
- 阅读服务条款（尤其是使用限制）
- 设置用量监控告警

### ❌ 避免的错误
- 虚报用途（如写"商业项目"实际是个人）
- 超出免费额度未及时升级
- 忘记更新应用状态
- 忽略API使用限制
- 未设置用量监控
