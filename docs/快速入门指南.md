# 🚀 战术地图快速入门指南

## 🎯 5分钟快速上手

### 第一步：启动项目
```bash
npm run dev
```
访问：`http://localhost:5173`

### 第二步：打开战术地图
点击导航栏中的 **"战术地图 v2"** 标签（带雷达图标 📡）

### 第三步：尝试基础操作

#### 1. 🖱️ 交互测试
- **悬停**：鼠标移到地图上的金色圆点（站点）上
- **观察**：鼠标变为十字准星，显示目标信息

#### 2. 🎯 点击测试
- **点击**：点击任意站点
- **观察**：
  - 弹出赛博朋克风格战术弹窗
  - 金色雷达波扩散动画
  - 左上角显示详细情报面板

#### 3. 📍 添加标记
- **点击**：地图空白处
- **结果**：自动添加自定义标记点

## 📊 数据导入实战

### 方法一：CSV 批量导入
1. 点击 **"情报中枢"** 标签
2. 准备 CSV 文件（格式：地名,网址）
3. 拖拽文件到上传区域
4. 等待自动解析和坐标定位

### 方法二：Google 智能检索
1. 在 **"Google 战术雷达检索"** 输入框输入地名
2. 选择类型（地标/车站/机场等）
3. 点击 **"锁定目标坐标"**
4. 自动添加到地图

## 🔧 常见问题解决

### ❌ 地图不显示
**原因**：Mapbox Token 未配置或无效
**解决**：
1. 检查 `.env` 文件中的 `VITE_MAPBOX_ACCESS_TOKEN`
2. 确认 Token 有效且未过期

### ❌ 样式异常
**原因**：Style URL 错误或网络问题
**解决**：
1. 确认 Style URL: `mapbox://styles/iopqwe51/cmnoq0jyc008501sg88f6en5z`
2. 检查网络连接

### ❌ 动画卡顿
**原因**：性能问题
**解决**：
1. 关闭其他浏览器标签
2. 降低地图缩放级别
3. 减少标记点数量

### ❌ API 限流
**原因**：请求过于频繁
**解决**：
1. 等待 1 分钟后重试
2. 使用批量导入功能
3. 利用缓存机制

## 💡 使用技巧

### 技巧1：批量导入地名
