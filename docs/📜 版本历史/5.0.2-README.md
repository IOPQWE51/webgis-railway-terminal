## 🚀 核心更新 (Core Updates)

### 1. 移动端战术抽屉系统 (Mobile Tactical Bottom Sheet)

#### 问题描述
在 v5.0.1 中，移动端点击地图标记时仍使用桌面端的 `#cyber-panel` 右侧浮窗，存在以下问题：
- 布局不适配移动端窄屏
- 无法单手操作
- 缺少手势交互支持

#### 解决方案
实现了全新的移动端战术抽屉组件 `TacticalBottomSheet.jsx`：

**特性:**
- ✅ 渐进式信息展示 (40% / 92% 两档高度)
- ✅ 琥珀金发光拖动指示器
- ✅ 手势跟随与磁吸锚点
- ✅ 深空毛玻璃背景
- ✅ 单手友好操作体验

**三档锚点:**
| 档位 | 高度 | 触发条件 |
|------|------|----------|
| 隐藏档 | 0% | 下拉 < 25% 或点击遮罩 |
| 侦察档 | 40% | 默认弹出高度 |
| 作战档 | 92% | 上拉 > 65% |

### 2. 手势劫持修复 (Touch Hijacking Fix)

#### 问题描述
移动端浏览器原生下拉刷新 (`pull-to-refresh`) 会与抽屉的拖动手势冲突，导致：
- 拖动时触发浏览器刷新
- 手势响应卡顿
- 用户体验糟糕

#### 解决方案
**技术手段:**
```javascript
// 1. 强制禁用浏览器默认触摸行为
touchAction: 'none'

// 2. 事件监听使用 passive: false
window.addEventListener('touchmove', handler, { passive: false })

// 3. 条件性阻止默认行为
if (e.cancelable) e.preventDefault()
```

**效果:**
- ✅ 完全消除浏览器原生下拉刷新干扰
- ✅ 恢复丝滑的拖动手感
- ✅ 物理阻尼效果自然

### 3. 数据流网关重构 (Data Gateway Refactor)

#### 问题描述
旧版 `cyberPanel.js` 直接操作 DOM，无法适配 React 组件系统。

#### 解决方案
重写 `cyberPanel.js`，实现设备检测与内容分发：

```javascript
export const openCyberPanel = (html) => {
    if (window.innerWidth < 1024) {
        // 移动端：发射自定义事件
        const event = new CustomEvent('openTacticalBottomSheet', { detail: html });
        window.dispatchEvent(event);
    } else {
        // PC端：保持原有 DOM 操作
        const panel = document.getElementById('cyber-panel');
        // ...
    }
};
```

**架构优势:**
- 解耦 DOM 操作与 React 组件
- 统一入口，便于维护
- 支持未来扩展其他设备类型

---

### 4. 视觉装甲增强 (Visual Armor Enhancement)

#### 问题描述
使用 Tailwind CSS 类名时，Vite JIT 编译可能出现缓存失效，导致样式不生效。

#### 解决方案
核心视觉元素采用**内联样式 (Inline Styles)**：

```javascript
// 琥珀金指示器 - 内联样式确保渲染
<div style={{
    width: '60px',
    height: '5px',
    backgroundColor: '#fbbf24',
    borderRadius: '999px',
    boxShadow: '0 0 12px #fbbf24'
}} />
```

**覆盖范围:**
- 拖动指示器
- 深色头部背景
- 分割线与边框
- 状态文字
