## 🚀 如何使用

### 1. 启动项目
```bash
npm run dev
```

### 2. 访问战术地图
- 打开浏览器访问 `http://localhost:5173`
- 点击"战术地图 v2"标签

### 3. 核心功能
- **交互**: 悬停/点击站点查看信息
- **导入**: 切换到"情报中枢"导入数据
- **标记**: 点击地图添加自定义点
- **航线**: 使用 API 绘制大圆航线

## 📞 技术支持

### 常见问题
1. **地图不显示**: 检查 Mapbox Token
2. **API 限流**: 等待 1 分钟重试
3. **动画卡顿**: 关闭其他标签页
4. **数据丢失**: 检查本地存储

### 调试工具
```javascript
// 性能检查
window.__tacticalMapTools.performanceCheck()

// 导出数据
window.__tacticalMapTools.exportData()

// 启动演示
window.__startAutoDemo()
```
